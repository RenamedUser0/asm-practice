int 0x80 - програмне переривання в Linux (архітектура x86, 32-bit).
Перед виконанням int 0x80:
EAX - номер системного виклику
EBX, ECX, EDX, ESI, EDI, EBP - параметри

Найбільш часто використовувані системні виклики:
sys_exit - Завершення програми.
EBX - код завершення

sys_write - вивід даних у файл або консоль.
EBX - дескриптор файлу
ECX - адреса буфера
EDX - довжина

sys_read - зчитування даних.
EBX - дескриптор(0 = stdin)
ECX - адреса буфера
EDX - кількість байтів

sys_open - Відкриття файлу.
EBX - ім'я файлу
ECX - режим відкриття
EDX - права доступу

sys_close - Закриття файлу.
EBX -файловий дескриптор
